# Sem 3 Lab Programs

This repository contains all the Java Web Technology Lab programs completed during Semester 3.

## Programs Included

1. Student Registration Form
   - HTML, JSP, Servlet, JDBC

2. Login Authentication System
   - JSP, Servlet, JDBC, Session Handling

3. Online Student Result System
   - JSP, Servlet, Database Connectivity

4. Employee Salary Calculator
   - Java Programming Concepts

5. Shopping Cart System
   - JSP, Servlet, JDBC

6. Library Management System
   - CRUD Operations using JDBC

7. Employee Management System
   - MVC Architecture
   - JDBC
   - Insert, Update, Delete, Search Operations

8. Online Feedback Management System
   - Form Validation
   - JDBC
   - Database Connectivity

9. Student Attendance System
   - JSP
   - Servlets
   - JDBC
   - Session Handling

10. Online Course Registration System
   - JSP
   - Servlet
   - JDBC

## Technologies Used

- Java
- JSP
- Servlets
- JDBC
- HTML
- CSS
- MySQL
- Apache Tomcat
- Eclipse IDE

## Database

MySQL is used for database connectivity in the JDBC-based applications.
